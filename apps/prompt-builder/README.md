# 🧪 Prompt Builder & Tester

> Compose high-quality, structured LLM prompts offline, then test or refine them with your own AI provider.

## What It Does

Prompt Builder & Tester helps you write better prompts. The Build tab turns structured fields — role, task, context, constraints, output format, and examples — into a single well-organized prompt you can copy and reuse, and it works entirely offline. The Test tab sends a system and user prompt to your configured LLM and shows the response, with character and word counts. You can save named prompt templates and optionally ask the AI to strengthen a prompt for you (meta-prompting). Everything stays in your browser.

## How To Use

1. Build a structured prompt (works fully offline).
2. Open Settings and add your AI provider API key to test prompts or improve them with AI (Claude by default, or any OpenAI-compatible endpoint).
3. Use "Improve with AI" to refine the assembled prompt, or switch to the Test tab to run a system + user prompt and view the model's response. Save useful prompts as templates and copy results with one click.

## Notes

- Runs entirely in the browser — no server of ours, no install, no account.
- Everything lives in a single self-contained index.html.
- Bring your own AI key: it is stored only in your browser, and your text is sent only to the provider you configure. A browser-stored key is visible to anyone with access to this device.
