# 🃏 Memory Match

> A fun, offline card-matching memory game in a single HTML file.

## What It Does
Memory Match lays out a grid of face-down cards, each hiding an emoji. Flip two cards at a time to find matching pairs — matched pairs stay revealed, while mismatches flip back over. Pick your difficulty (4×4, 6×4, or 6×6), race the clock, and try to clear the board in as few moves as possible. Your best score for each difficulty is remembered between sessions.

## How To Use
1. Open `index.html` in any modern web browser (double-click it — no server needed).
2. Choose a difficulty from the dropdown, then click **New Game**.
3. Click any card to flip it, then click a second card to look for its match.
4. Matching pairs stay face-up; mismatches flip back after a moment.
5. Clear every pair to win — your time and move count appear, along with any new best score. Click **Play Again** to go again.

## Notes
- Runs entirely in the browser — no server, no install, no account.
- Everything lives in a single self-contained index.html.
- Best scores are saved locally in your browser.
