# 📝 Text Summarizer
> Paste any text and get a clean summary in the format and length you choose — powered by your own AI key.

## What It Does
Text Summarizer turns long source text into a concise summary using a large language model you connect via your own API key. Paste text (or import a `.txt`/`.md` file), pick a format (paragraph, bullet points, TL;DR one-liner, or key takeaways), choose a length, and optionally tell it what to focus on. It shows live word and character counts for your input and the generated summary, and lets you copy the result with one click.

## How To Use
1. Open Settings and add your AI provider API key (Claude by default, or any OpenAI-compatible endpoint).
2. Paste or type your source text, or use "Import .txt / .md" to load a file.
3. Choose a summary format, a length, and (optionally) a "focus on…" topic.
4. Click "Summarize" and wait for the result to render.
5. Click "Copy" to copy the summary to your clipboard.

## Notes
- Runs entirely in the browser — no server of ours, no install, no account.
- Everything lives in a single self-contained index.html.
- Bring your own AI key: it is stored only in your browser, and your text is sent only to the provider you configure. A browser-stored key is visible to anyone with access to this device.
