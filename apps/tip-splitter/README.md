# 🧾 Tip Splitter
> Split any bill, add a tip, and see exactly what each person owes — instantly and offline.

## What It Does
Tip Splitter takes your bill amount, a tip percentage, and the number of people, then shows the tip, the total, and the per-person amount in big, easy-to-read numbers that update live as you type. It can round each person's share up to the nearest dollar, calculate the tip on the pre-tax amount, and handle uneven splits when some people pay more than others.

## How To Use
1. Enter the bill amount.
2. Pick a tip percentage with the quick buttons (10/15/18/20/25) or type a custom %.
3. Set the number of people with the big + / − steppers.
4. (Optional) Turn on "Round up per person" to round each share up to the next dollar.
5. (Optional) Turn on "Tip on pre-tax amount" and enter the tax to exclude it from the tip base.
6. (Optional) Turn on "Uneven split" to give each person their own share of the bill.
7. Read the results — per-person amount, tip, and total update automatically.
