# 🗒️ Notes

> A fast, private, offline note-taking app that lives in a single HTML file.

## What It Does

Notes lets you write, organize, and search your notes entirely in your browser. It supports full CRUD with debounced auto-save, live text search, sorting by last-edited or title, tags for filtering, and pinning important notes to the top. You can export all of your notes as a JSON backup, import them back (merging or replacing), and export any single note as a plain `.txt` file. Nothing ever leaves your device.

## How To Use

1. Open `index.html` in any modern browser (double-click works — no server needed).
2. Click **＋ New note** (or press Ctrl/Cmd+N) and start typing — your title, body, and tags auto-save as you go.
3. Add comma-separated tags, then click a tag chip in the sidebar to filter; use the search box to find text and the **Sort** menu to reorder.
4. Use the toolbar to **Pin** a note to the top, **Duplicate** it, export it as **.txt**, or **Delete** it (with confirmation).
5. Use **Export** to download a JSON backup of everything, and **Import** to merge or replace from a backup file.

## Notes

- Runs entirely in the browser — no server, no install, no account.
- Everything lives in a single self-contained index.html.
- All notes are stored locally in your browser; export a JSON backup to keep them safe.
