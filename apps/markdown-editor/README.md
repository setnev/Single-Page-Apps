# 📝 Markdown Editor

> A fast, offline markdown editor with live preview and local multi-document storage.

## What It Does

Markdown Editor is a single-file web app for writing markdown with an instant, side-by-side rendered preview. It ships with its own small markdown parser (headings, bold/italic, inline code, code fences, blockquotes, ordered & unordered lists with nesting, links, images, horizontal rules, and paragraphs), a formatting toolbar, word/character counts, and export to `.md` or a standalone `.html` file. You can keep multiple documents, switch between them, and everything is saved automatically to your browser.

## How To Use

1. Open `index.html` in any modern browser (double-click works — no server needed).
2. Type markdown in the left editor; the rendered result appears live on the right.
3. Use the toolbar buttons (or Ctrl/Cmd+B and Ctrl/Cmd+I) to insert or wrap formatting.
4. Manage documents in the left sidebar: **+ New**, **Rename**, **Delete**, and click any doc to switch. Your work autosaves as you type.
5. Export with the top-bar buttons: download `.md` or a standalone `.html`, or copy the markdown / rendered HTML to your clipboard.

## Notes

- Runs entirely in the browser — no server, no install, no account.
- Everything lives in a single self-contained index.html.
- Documents are saved locally in your browser (localStorage); the preview sanitizes HTML.
