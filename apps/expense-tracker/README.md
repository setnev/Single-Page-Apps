# 💰 Expense & Budget Tracker

> A private, offline, single-file app for tracking income, expenses, and monthly budgets.

## What It Does
Expense & Budget Tracker lets you log income and expenses with categories, dates, and notes, then see your current balance, total income, and total expenses for any month. It visualizes spending with a hand-drawn donut chart and per-category breakdown, supports an optional monthly budget with progress and over-budget warnings, and lets you search and filter your transactions. You can set a currency symbol and export or import your data as CSV/JSON backups.

## How To Use
1. Open `index.html` in any modern browser (double-click it — no server needed).
2. Click **＋ Add transaction** to log income or expense: enter the amount, pick income/expense, choose or add a category, set the date, and optionally add a note.
3. Use the **‹ / ›** month switcher to move between months and review that month's balance, income, and expenses.
4. Click **Set budget** to add an optional monthly budget; the progress bar turns yellow as you approach it and red when you go over.
5. Search, filter by type/category, and click the **✎** icon on any transaction to edit or delete it.
6. Open **⚙ Settings** to change the currency symbol or manage categories, and use **Export CSV / Export JSON / Import JSON** to back up or restore your data.

## Notes
- Runs entirely in the browser — no server, no install, no account.
- Everything lives in a single self-contained index.html.
- All financial data stays on your device; export a backup to keep it safe.
